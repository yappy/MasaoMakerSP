このたびは、 まさおコンストラクション を、ダウンロードしていただき、
ありがとうございます。


[ 内容について ]

このゲームは、 InternetExplorer4.0 以上、 NetscapeNavigator4.5 以上で
動作する JavaApplet です。転載可能、許可不要のフリーソフトであり、
マップや、グラフィックやパラメータを変更すれば、
あなたのホームページに、オリジナルに近いゲームが登場します。


[ ゲームで遊ぶ ]

game1.html をダブルクリックする等して、ブラウザで実行して下さい。
game2.html 、 game3.html などもあります。

[ グラフィックの変更] 

以下の *.gif ファイルを、描き変えて下さい。

title.gif         タイトル画面
gameover.gif      ゲームオーバー画面
ending.gif        エンディング画面
pattern.gif       キャラクター等のグラフィック
chizu.gif         地図画面の背景

キャラクターは、 320 * 640 ドットの絵を、 32 * 32 ドット 200 枚 に切って
使っています。 pattern.bmp を参考にして下さい。
内容は、見てもらえれば、わかると思います。
ここでは、カラーインデックス 5 （暗い紫色）を透明色に使っています。


[ ゲーム内容の変更 ]

マップや、制限時間の設定は、 game1.html という、
ＨＴＭＬファイルを、メモ帳などの、テキストエディタで編集して行います。
 PARAM タグにある "" 内の文字を変更します。
詳しくは、そちらを見て下さい。


[ ゲームの公開 ]

以下の、 7 個のファイルを、同じフォルダに、置いて下さい。

ending.gif
game1.html  または、 game2.html 、 game3.html など。
gameover.gif
mc_c.zip
pattern.gif
title.gif
chizu.gif          地図画面を使用する場合


 mc_c.zip は、解凍しない方が、プログラムの起動が高速です。
動作しない場合は、解凍してみて下さい。
