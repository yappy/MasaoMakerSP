まさおコンストラクション をダウンロードしていただき、
ありがとうございます。
当ソフトは、アクションゲームを簡単に作れるツールです。
マップや、グラフィックやパラメータを変更すれば、
あなたのホームページに、オリジナルに近いゲームが登場します。


[ ファイル構成 ]

readme.txt    このファイル
mc_c.zip      ゲームのプログラム本体
pattern.gif   ゲーム中で使われる画像
title.gif     同上
gameover.gif  同上
ending.gif    同上
mapchip.gif   同上
haikei.gif    同上
pattern.bmp   画像編集時の参考用
game1.html    サンプルゲーム
game_mt.html  背景レイヤー対応サンプルゲーム

mt3.exe       専用マップエディタ  まさおツール Version 3.22
parts.bmp     マップエディタ用の画像
mapchip.bmp   同上
mt_data.txt   マップエディタのHTML書き出しに必要
sample.fmf    サンプルマップデーター

omake.zip     Version 2.9 に付属していたサンプゲームと、効果音ファイル。
              必要に応じて解凍し、同じフォルダに置いて下さい。

まさおツールは、 Visual C# 2005 で作られており、
Microsoft .NET Framework 2.0 がインストールされた環境で動作します。


[ ゲームを起動する ]

game1.html をダブルクリックするなどして、ブラウザで実行して下さい。
game_mt.html もあります。


[ グラフィックの変更] 

以下の *.gif ファイルを、描き変えて下さい。

title.gif         タイトル画面
gameover.gif      ゲームオーバー画面
ending.gif        エンディング画面
pattern.gif       キャラクター等のグラフィック
mapchip.gif       背景レイヤーのマップチップ
haikei.gif        背景画像
chizu.gif         地図画面の背景

キャラクターは、 320*640 ドットの画像を、
32*32 ドット 200 枚に切って使っています。
pattern.bmp を参考にして下さい。
内容は、見てもらえれば、わかると思います。
暗い紫色 (Red=128 Green=0 Blue=128) を透明色に使っています。

透過 GIF 形式のファイルを使用していますが、
Sun Microsystems の JavaVM では、
アルファチャンネル付き PNG 形式でも動作する場合があります。


[ ゲーム内容の変更 ]

マップや、制限時間の設定は、 game1.html という、
HTML ファイルを、メモ帳などの、テキストエディタで編集して行います。
param タグにある "" 内の文字を変更します。
詳しくは、そちらを見て下さい。


[ ゲームの公開 ]

以下のファイルを、同じフォルダに置いて下さい。

game1.html  または、 game_mt.html など。
mc_c.zip
pattern.gif
title.gif
gameover.gif
ending.gif
mapchip.gif         背景レイヤーを使用する場合
haikei.gif          同上
chizu.gif           地図画面を使用する場合
*.au                効果音を使用する場合は効果音ファイル全部

mc_c.zip は、解凍しない方が、プログラムの起動が高速です。
動作しない場合は、解凍してみて下さい。


[ 効果音ファイルについて ]

当ソフトでは、 au 形式の効果音を使用しています。
好きな効果音を使いたい場合は、 WAVE(*wav) 形式のファイルを自作するか、
フリー素材で入手するなどしてから、
WavToAu などのソフトで、 au 形式に変換して下さい。

SunMicrosystems の JavaVM では、WAVE 形式でも動作する場合があります。


[ 背景レイヤー ]

まさおコンストラクション Version 3.0 では、
game_mt.html のマップデーターの下に、背景レイヤーデーターが追加されました。
256種類のマップチップが、2桁の16進数で表現されます。
0番と255番は透明なチップです。
編集用のマップチップ画像は、 mapchip.bmp とし、
512*512 ドットを、32*32 ドット 256 枚に切って使っています。
ゲーム中のマップチップ画像は、 mapchip.gif を使用しています。

下から通れるブロック ([) と、ハシゴブロック (]) が追加されました。
背景レイヤーを表示すると、その２種と、
ブロック１０ (j) と、水中ブロック (4) が透明表示になります。


[ Platinum で編集 ]

まさおツールで保存される *.fmf 形式のファイルは、
汎用マップエディタ Platinum
HyperDevice software = http://www.hyperdevice.net/
Vector ダウンロード = http://www.vector.co.jp/soft/win95/game/se231004.html
で、読み書きできます。
Platinum を起動したら、ファイルメニュー > 読み込み > Fixed MapData File を選択して、
当キットに付属の sample.fmf を読み込みます。
ウィンドウメニューからパーツウィンドウを開き、その左上にあるボタンで、
mapchip.bmp と parts.bmp を開きます。
ウィンドウメニューからレイヤーウィンドウを開き、
レイヤー１に parts.bmp 、レイヤー０に mapchip.bmp の画像を、設定します。
ビューメニューから、透過処理を行うを、チェックします。
透過色の指定で、赤128 緑0 青128 と入力します。
これで編集の準備完了。
名前を付けて保存で *.ppj プロジェクトファイルを作れば、次回からその設定で開けます。
マップの編集が終わったら、ファイルメニューから *.fmf 形式で、書き出し、
それをまさおツールで開き、HTML書き出しで、完成です。
